HTML Head Tag

The head Element

The head element is a container for metadata (data about data) and is placed between the html tag and body tag.

The title Element

The title element defines the title of the document. It is shown in the browser's title bar or in a page's tab.

The page title is used by search engine algorithms to decide the order when listing pages in search results.

The link Element

The link element defines the relationship between the current document and external resource. The link tag is most often used to link to external style sheets.

The meta Element

The meta element is typically used to specify the character set, page description, keywords, author of the document and viewport settigs. The metadata will not be displayed on the page, but is used by browsers (how to display content), by search engines for keywords and other web services.

Some of the meta tags are:

  • Defines the character set used:

    <meta charset="UTF-8">

  • Defines the viewport to make your website look good on all devices:

    <meta name="viewport" content="width=device-width">

  • Defines keywords for search engines:

    <meta name="keywords" content="HTML, CSS, JavaScript">

  • Defines the author of a web page:

    <meta name="author" content="John Doe">
